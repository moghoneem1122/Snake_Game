## 🛠️ How to Build and Run

To run this project, you need to have the following installed:

* **MinGW (GCC compiler for Windows)**
* **raylib library**

---

### ⚙️ Build Instructions

1. Open the project in **VS Code**
2. Open the terminal inside VS Code
3. Navigate to the build folder:

```
cd build
```

4. Compile the project using:

```
mingw32-make
```

---

### ▶️ Run the Game

After building, an executable file (`game.exe`) will be created.

* Navigate to the **game folder**
* Open `game.exe` to start the game

---

### 📌 Notes

* Make sure `raylib` is properly installed and linked
* Ensure `mingw32-make` is available in your system PATH
